# Synthetic evidence-contract incident response

Stop immediately if the system under test observes future material, the oracle,
a credential, source content in a metadata-only log, or any network or telemetry
path. Mark the attempt invalid and do not resume under the same run identity.

For this public fictional pack, remove the isolated workspace, verify that no
process or mount remains, and append only the incident category and disposition
to the failure diary. No private data is authorized for this pack. A later
private run uses its own approved notification, retention, backup, and purge
procedure.
