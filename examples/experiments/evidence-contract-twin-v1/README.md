# Evidence-contract synthetic twin v1

This frozen, public, fully fictional pack exercises the evidence-contract shape
required by G1. It was authored from generic experimental task classes without
access to a private episode. It contains no transformed, summarized, hashed, or
otherwise derived private material.

Four sources reveal a brief, a decision, a stale copied status, and an
authoritative correction. Eight frozen tasks cover:

- an answer available at an early horizon;
- required abstention until later evidence arrives;
- correction that retains material counterevidence;
- terminal abstention at a declared closed-corpus boundary.

The primary baseline uses the same visible Markdown bytes plus `rg`, with the
same horizon, task timeout, query budget, response contract, and assistance
policy. GraphTruth-only preparation and maintenance count as capture tax.

This is a G1 evidence-contract fixture, not a completed experiment. It does not
bind executable code, configuration, environment, or a runtime report. No
evaluated reveal, baseline, scoring pass, decision, sandbox rehearsal, or
private-data admission has occurred. Those remain M2 work under a new final run
card that binds the exact runtime identity.

`pack-lock.json` closes this directory and is externally anchored by the Git
commit that publishes it. `disclosure-review.md` records the public disclosure
check and the private-episode comparison that must remain pending until the
closed owner and reviewer phase.

Validate the frozen public bytes from the repository root:

```sh
./tooling/preflight --twin
```
